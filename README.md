# TechWeb_P2  - NBA CALENDAR

Nesse web project você pode criar sua conta no NBA calendar para registrar jogos que você quer assistir dos principais times da liga e ainda avisar amigos sobre eles.
Caso também tenha perdido o jogo, você pode ver um resumo de como foi.

-Setup: 

Para utilizar a função de aviso via Whatsapp você e seus amigos devem enviar uma mensagem "join thistle-pufin" para +1(415)523-8886.
Formato do número: +551112345678 

-Descrição: Esse projeto possui uma arquitetura Spring MVC em REST e utiliza uma API de WebScrapper para puxar dados do site oficial da NBA e também uma API da Twilio para envio de mensagem via whatsapp.
